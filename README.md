# Playloom

Playloom is a visual editor for infinite game diagrams.

It uses a structured JSON format called Playscript and exports ordinary TikZ.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is written to `dist`.

## Deploy

For Vercel, use the default Vite settings:

- Build command: `npm run build`
- Output directory: `dist`
